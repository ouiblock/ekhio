# ☀️ Ekhio

> **Grandir, un jour à la fois.** — *Growing up, one day at a time.*

Ekhio is a free, open-source parenting companion app for children aged **3 to 18 years old**, built on neuroscience, developmental psychology, and child psychiatry.

---

## 🤖 What is Ekhio?

Ekhio is a family-centered reward system that grows with your child. Unlike generic chore apps, Ekhio adapts its behaviors, rewards, and interface to each developmental stage — from a 3-year-old learning to share, to a 17-year-old building autonomy.

The name **Ekhio** means **Sun** in Basque (Euskara) — a nod to our roots in the Pays Basque, France.

---

## ✨ Key Features

- 🎯 **Age-adapted** — 5 developmental stages (3-6 / 7-10 / 11-13 / 14-16 / 17-18)
- 🧠 **Science-based** — Piaget, Deci & Ryan (SDT), positive reinforcement research
- 🔒 **Zero data collection** — works fully offline via localStorage, no login, no account
- 🎮 **Mini-game** — unlockable robot runner game (10 stars required)
- 👨‍👩‍👧 **Parent mode** — manage behaviors, rewards, daily limits
- 📖 **History** — full earn/spend reward history
- 📱 **PWA** — installable on iOS and Android as a home screen app
- 🌍 **Open source** — MIT + Commons Clause (non-commercial)

---

## 🧠 Scientific Foundations

| Domain | Reference |
|--------|-----------|
| Neuroscience | Limbic system vs prefrontal cortex development |
| Psychology | Self-Determination Theory — Deci & Ryan (1985) |
| Pedagogy | Cognitive stages — Jean Piaget |
| Child psychiatry | Positive reinforcement — Skinner, Lepper (over-justification effect) |
| Emotional regulation | Co-construction of rules from age 11 |

## 🚀 Getting Started

### Use directly
Visit the live app or open `index.html` in any modern browser.

### Self-host
```bash
git clone https://github.com/ouiblock/ekhio.git
cd ekhio
# Open index.html in your browser — no build step needed
```

### Deploy on Netlify
1. Fork this repo
2. Connect to Netlify → Deploy
3. Build command: (empty) · Publish directory: `.`
